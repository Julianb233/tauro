<task type="auto">
  <name>Task 2: Create PropertyMap and ImageGallery components</name>
  <files>src/components/PropertyMap.tsx, src/components/ImageGallery.tsx</files>
  <action>
    **Create `src/components/PropertyMap.tsx`** -- a "use client" component using react-map-gl declarative API:

    IMPORTANT: Import from "react-map-gl/mapbox" (NOT default import). This is required for mapbox-gl v2+.

    ```tsx
    "use client";

    import { useState, useCallback } from "react";
    import Map, { Marker, Popup, NavigationControl } from "react-map-gl/mapbox";
    import "mapbox-gl/dist/mapbox-gl.css";
    import { Property, formatPrice } from "@/data/properties";
    ```

    Props interface:
    ```tsx
    interface PropertyMapProps {
      properties: Property[];
      onPropertyClick?: (slug: string) => void;
      center?: [number, number];  // [lng, lat]
      zoom?: number;
      singleMarker?: boolean;     // For detail page (single property, no popup)
    }
    ```

    Implementation:
    - Check if `process.env.NEXT_PUBLIC_MAPBOX_TOKEN` exists. If empty/undefined, render a styled fallback div with text "Map unavailable -- configure Mapbox token" using MapPin icon from lucide-react, matching the dark brand style.
    - Use `useState<Property | null>(null)` for `popupInfo`
    - Default center: `[-117.16, 32.78]` (matches existing property data coordinates)
    - Default zoom: `10` for multi-property, caller passes `14` for single
    - Render `<Map>` with:
      - `mapboxAccessToken={process.env.NEXT_PUBLIC_MAPBOX_TOKEN}`
      - `initialViewState={{ longitude: center[0], latitude: center[1], zoom }}`
      - `style={{ width: "100%", height: "100%" }}`
      - `mapStyle="mapbox://styles/mapbox/dark-v11"`
    - Inside `<Map>`:
      - `<NavigationControl position="top-right" />`
      - Map over `properties` to render `<Marker>` for each:
        ```tsx
        <Marker key={p.id} longitude={p.lng} latitude={p.lat} anchor="bottom">
          <button
            onClick={(e) => {
              e.stopPropagation();
              if (singleMarker) return;
              setPopupInfo(p);
            }}
            className="rounded-full bg-midnight px-2 py-1 text-xs font-bold text-gold shadow-md transition-colors hover:bg-gold hover:text-near-black"
          >
            {formatPrice(p.price)}
          </button>
        </Marker>
        ```
      - When `popupInfo` is set and NOT singleMarker, render `<Popup>` with a mini property card:
        ```tsx
        <Popup
          longitude={popupInfo.lng}
          latitude={popupInfo.lat}
          anchor="bottom"
          offset={25}
          onClose={() => setPopupInfo(null)}
          closeOnClick={false}
        >
          <div className="max-w-[200px] p-2">
            <p className="font-heading text-sm font-bold text-near-black">{popupInfo.address}</p>
            <p className="text-xs text-gray-600">{popupInfo.city}, {popupInfo.state}</p>
            <p className="mt-1 font-bold text-near-black">{formatPrice(popupInfo.price)}</p>
            <p className="text-xs text-gray-500">{popupInfo.beds} BD | {popupInfo.baths} BA | {popupInfo.sqft.toLocaleString()} SF</p>
            {onPropertyClick && (
              <button
                onClick={() => onPropertyClick(popupInfo.slug)}
                className="mt-2 w-full rounded bg-near-black px-2 py-1 text-xs font-semibold text-gold hover:bg-midnight"
              >
                View Details
              </button>
            )}
          </div>
        </Popup>
        ```

    Do NOT use next/dynamic or ssr:false -- the "use client" + react-map-gl is the correct App Router approach. Do NOT use raw mapbox-gl API for markers -- use react-map-gl declarative `<Marker>` and `<Popup>` components.

    **Create `src/components/ImageGallery.tsx`** -- a "use client" lightbox gallery:

    Imports:
    ```tsx
    "use client";

    import { useState } from "react";
    import Lightbox from "yet-another-react-lightbox";
    import Counter from "yet-another-react-lightbox/plugins/counter";
    import Fullscreen from "yet-another-react-lightbox/plugins/fullscreen";
    import Thumbnails from "yet-another-react-lightbox/plugins/thumbnails";
    import Zoom from "yet-another-react-lightbox/plugins/zoom";
    import "yet-another-react-lightbox/styles.css";
    import "yet-another-react-lightbox/plugins/counter.css";
    import "yet-another-react-lightbox/plugins/thumbnails.css";
    ```

    Props: `images: string[]`, `address: string`

    State: `open: boolean` (false), `index: number` (0)

    Render a thumbnail grid:
    - First image spans full width: `aspect-[16/9] max-h-[500px] lg:aspect-[21/9]` with cursor-pointer, rounded-xl, overflow-hidden. Clicking opens lightbox at index 0.
    - Below: row of up to 4 smaller thumbnails (`h-16 w-24 flex-shrink-0 rounded-lg`). Active thumbnail gets `border-2 border-gold`, others `border-transparent opacity-60 hover:opacity-100`.
    - If >5 images total, last visible thumbnail shows "+N more" overlay (dark overlay with white text).
    - Clicking any thumbnail opens the lightbox at that index.

    Lightbox component:
    ```tsx
    <Lightbox
      open={open}
      close={() => setOpen(false)}
      index={index}
      slides={images.map((src) => ({ src }))}
      plugins={[Counter, Fullscreen, Thumbnails, Zoom]}
      carousel={{ finite: true }}
      styles={{ container: { backgroundColor: "rgba(0, 0, 0, 0.95)" } }}
    />
    ```

    Use `{/* eslint-disable-next-line @next/next/no-img-element */}` before img tags (matching existing codebase pattern -- project uses Unsplash URLs, not local images).
  </action>
