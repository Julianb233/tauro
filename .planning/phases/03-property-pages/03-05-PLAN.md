**Props interface**:
```typescript
interface PropertyMapProps {
  properties: Property[];
  onPropertyClick?: (slug: string) => void;
  center?: [number, number]; // [lng, lat] -- default [-75.16, 39.95] (Philadelphia)
  zoom?: number; // default 10
  singleMarker?: boolean; // For detail page -- no popup on click
}
```

**Token handling**:
- Read `process.env.NEXT_PUBLIC_MAPBOX_TOKEN`
- If missing or equals `"pk.placeholder_token_replace_me"`, render a fallback UI: flex centered container with MapPin icon (h-10 w-10 text-gold) and "Map unavailable -- configure Mapbox token" message in gold text, on bg-near-black/80

**Map implementation** using `react-map-gl/mapbox`:
- `<Map>` component with `mapboxAccessToken`, `initialViewState` (longitude, latitude, zoom), full width/height style, `mapStyle="mapbox://styles/mapbox/dark-v11"`
- `<NavigationControl position="top-right" />` for zoom buttons
- For each property, render `<Marker longitude={p.lng} latitude={p.lat} anchor="bottom">` containing a styled button showing `formatPrice(p.price)` -- styled as `rounded-full bg-midnight px-2 py-1 text-xs font-bold text-gold shadow-md transition-colors hover:bg-gold hover:text-near-black`
- On marker click: if not singleMarker, call `e.stopPropagation()` and set `popupInfo` state to that property. If singleMarker, return early.
- `<Popup>` (when popupInfo is set and not singleMarker): `anchor="bottom" offset={25} onClose={() => setPopupInfo(null)} closeOnClick={false}`. Content: property address (font-heading, text-near-black), city/state, price (bold), beds/baths/sqft, and "View Details" button that calls `onPropertyClick(popupInfo.slug)`. Max-width 200px with padding.

**State**: `popupInfo: Property | null` initialized to null
