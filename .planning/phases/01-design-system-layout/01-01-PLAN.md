<task type="auto">
  <name>Task 2: Add micro-interaction CSS utility classes</name>
  <files>src/app/globals.css</files>
  <action>
APPEND the following CSS at the end of globals.css, after the `.font-label` rule added in Task 1. Do NOT modify any existing content.

**Gold shimmer hover effect (pseudo-element overlay on hover):**
```css
/* ===== Tauro Micro-interactions (BRAND-03) ===== */

@keyframes shimmer-sweep {
  0% { background-position: -100% 0; }
  100% { background-position: 200% 0; }
}

.shimmer-gold {
  position: relative;
  overflow: hidden;
}

.shimmer-gold::after {
  content: '';
  position: absolute;
  inset: 0;
  background: linear-gradient(
    45deg,
    transparent 25%,
    rgba(201, 168, 76, 0.15) 50%,
    transparent 75%
  );
  background-size: 250% 100%;
  background-position: -100% 0;
  transition: background-position 0s ease;
  pointer-events: none;
}

.shimmer-gold:hover::after {
  background-position: 200% 0;
  transition-duration: 1.5s;
}
```

**3D card tilt (CSS-only hover):**
```css
.card-tilt {
  transition: transform 0.3s ease;
  transform-style: preserve-3d;
}

.card-tilt:hover {
  transform: perspective(800px) rotateX(2deg) rotateY(-3deg) scale(1.02);
}
```

**Diagonal wipe reveal animation:**
```css
@keyframes diagonal-wipe-in {
  from { clip-path: polygon(0 0, 0 0, 0 100%, 0 100%); }
  to { clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%); }
}

.diagonal-wipe {
  animation: diagonal-wipe-in 0.6s ease-out forwards;
}
```

**Slash/swipe reveal:**
```css
@keyframes slash-reveal-in {
  from { clip-path: polygon(0 0, 0 0, 0 100%, 0 100%); }
  to { clip-path: polygon(0 0, 115% 0, 100% 100%, 0 100%); }
}

.slash-reveal {
  animation: slash-reveal-in 0.7s cubic-bezier(0.25, 0.46, 0.45, 0.94) forwards;
}
```

**Diagonal wipe transition (hover/class-toggle variant):**
```css
.diagonal-wipe-transition {
  clip-path: polygon(0 0, 0 0, 0 100%, 0 100%);
  transition: clip-path 0.6s ease-out;
}

.diagonal-wipe-transition.active,
.diagonal-wipe-transition:hover {
  clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%);
}
```

**Reduced motion accessibility:**
```css
@media (prefers-reduced-motion: reduce) {
  .shimmer-gold::after,
  .card-tilt,
  .diagonal-wipe,
  .slash-reveal,
  .diagonal-wipe-transition {
    animation: none !important;
    transition: none !important;
    transform: none !important;
    clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%) !important;
  }
}
```

Use plain CSS classes (not @utility directives) since pseudo-class selectors like `:hover` and pseudo-elements like `::after` work more reliably as plain CSS. These classes will still work when applied via className.
