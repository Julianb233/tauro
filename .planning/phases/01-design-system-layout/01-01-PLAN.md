  **src/components/ui/gold-shimmer.tsx:**
  - "use client" directive
  - Props: `children: React.ReactNode`, `className?: string`, spread `HTMLAttributes<HTMLDivElement>`
  - Renders `<div className={cn("gold-shimmer", className)} {...props}>`
  - Wraps children in `<span className="relative z-2">` so content stays above the shimmer pseudo-element
  - Named export `GoldShimmer`

  **src/components/ui/tilt-card.tsx:**
  - "use client" directive
  - Props: `children: React.ReactNode`, `className?: string`, `maxTilt?: number` (default 8), spread `HTMLAttributes<HTMLDivElement>`
  - Uses `useRef<HTMLDivElement>` for the card element
  - Uses `useState` for transform string (initial: `"perspective(800px) rotateX(0deg) rotateY(0deg)"`) and transition timing (initial: `"transform 0.15s ease-out"`)
  - `onMouseMove` handler (wrapped in `useCallback`):
    - Get card bounding rect
    - Calculate center and half dimensions
    - `rotateY = ((clientX - centerX) / halfWidth) * maxTilt`
    - `rotateX = -((clientY - centerY) / halfHeight) * maxTilt`
    - Set transform: `perspective(800px) rotateX(${rotateX.toFixed(2)}deg) rotateY(${rotateY.toFixed(2)}deg)`
    - Set transition: `"transform 0.15s ease-out"`
  - `onMouseLeave` handler (wrapped in `useCallback`):
    - Reset transform to neutral
    - Set transition: `"transform 0.4s ease-out"` (slower return)
  - Apply via inline `style={{ transform, transition, willChange: "transform", transformStyle: "preserve-3d" }}`
  - Named export `TiltCard`

  Both use `cn()` from `@/lib/utils`.
  </action>
