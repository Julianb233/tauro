  **Create src/hooks/use-scrolled.ts:**

  A "use client" hook that tracks whether the user has scrolled past a threshold:
  - `useState(false)` for scrolled state
  - `useEffect` with passive scroll listener on window
  - Returns true when `window.scrollY > threshold` (default 50)
  - Calls handler immediately on mount to check initial position
  - Cleans up listener on unmount
  - Named export `useScrolled`

  **Create src/components/navbar.tsx:**

  "use client" component. Imports: useState, useEffect from react; Link from next/link; Menu, X, Phone from lucide-react; cn from @/lib/utils; Logo from @/components/logo; useScrolled from @/hooks/use-scrolled.

  Navigation links array (exactly these 5, no others):
  ```ts
  const navLinks = [
  ```
