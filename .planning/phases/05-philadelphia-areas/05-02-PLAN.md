- Icons from lucide-react: ChevronRight

**src/components/market-stats.tsx** -- `MarketStats` component:
Props: `{ stats: Neighborhood['stats']; neighborhoodName: string }`.
Renders:
- Section with gold label "Market Data", h2 "{neighborhoodName} by the Numbers"
- 4-column responsive grid (`sm:grid-cols-2 lg:grid-cols-4`)
- Four stat cards: medianPrice (DollarSign), avgPricePerSqft (TrendingUp), avgDaysOnMarket (Clock), inventoryLevel (BarChart3)
- Each card: `rounded-xl border border-border bg-card p-6 text-center`
- Icon in `text-muted-foreground`, value in `text-gold font-heading text-2xl font-bold`, label in `text-sm text-muted-foreground`
- Background: `bg-midnight` with `border-t border-border/40`

**src/components/area-listings.tsx** -- `AreaListings` component:
Props: `{ neighborhoodName: string; propertyFilter: string }`.
Renders:
- Imports `properties` from `@/data/properties` and `PropertyCard` from `@/components/PropertyCard`
- Filters properties where `property.city.toLowerCase() === propertyFilter.toLowerCase()`
- If matches found: responsive grid (`sm:grid-cols-2 lg:grid-cols-3`) of PropertyCard components
- If no matches: empty state card with MapPin icon, "New listings in {neighborhoodName} coming soon" heading, description text, gold "Get Notified" button linking to /contact with ArrowRight icon
- Gold label "Available Listings", h2 "Properties in {neighborhoodName}"
- Background: `bg-near-black`

All components follow existing site patterns: `font-heading` for headings, `font-label` for uppercase labels, `text-gold` for accents, dark theme throughout.
