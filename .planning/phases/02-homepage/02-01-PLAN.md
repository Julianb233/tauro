# Plan: Cinematic Hero Section
**Phase:** 2 -- Homepage
**Plan ID:** 02-01
**Status:** PENDING
**Depends On:** Phase 1 complete (design system, layout shell, nav, footer)
**Wave:** 1
**Autonomous:** true

## Goal

Build the cinematic full-bleed hero section as a standalone component and wire it into the homepage. The hero fills the viewport with an image background (video-ready structure), layered gradient overlay, animated headline with gold accent, subheadline, and a functional search bar that routes to /properties with query params. Includes a stats bar beneath the hero and a scroll-down hint animation.

This is the first thing visitors see -- it must feel premium, cinematic, and conversion-focused. Think SERHANT or The Agency level visual impact.

## Requirements Addressed

- **HOME-01**: Cinematic full-bleed hero section with video/image background, gradient overlay, and search bar
