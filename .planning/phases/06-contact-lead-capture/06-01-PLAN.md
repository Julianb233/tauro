<task type="auto">
  <name>Task 1: Create ContactForm component with validation and API submission</name>
  <files>src/components/contact-form.tsx</files>
  <action>
Create `src/components/contact-form.tsx` as a CLIENT component ("use client").

Import `LeadPayload` type from `@/app/api/leads/route` for type safety.

**State management:**
- `formData` state: `{ firstName, lastName, email, phone, message }` (all strings, initially empty)
- `formState` state: `"idle" | "submitting" | "success" | "error"`
- `errors` state: `Record<string, string>` for field-level validation errors

**Validation function** (runs on submit):
- `firstName`: required, min 2 chars
- `lastName`: required, min 2 chars
- `email`: required, must match `/^[^\s@]+@[^\s@]+\.[^\s@]+$/`
- `phone`: required, min 10 chars
- Returns boolean (valid/invalid), sets field errors in state

**Submit handler:**
1. Validate all fields, return early if invalid
2. Set formState to "submitting"
3. POST to `/api/leads` with body: `{ type: "contact", ...formData }` as LeadPayload
4. On success (response.ok): set formState "success", reset form data
5. On error: set formState "error"

**Render:**
- Two-column name fields (firstName + lastName) side by side on sm+
- Email field (type="email")
- Phone field (type="tel")
- Message textarea (4 rows)
- All inputs: `bg-near-black border border-border rounded-lg px-4 py-3 text-foreground placeholder:text-muted-foreground focus:border-gold focus:ring-1 focus:ring-gold outline-none transition-colors`
- Field error text: `text-red-400 text-xs mt-1`
- Submit button: `w-full bg-gold text-near-black font-semibold py-3 rounded-lg hover:bg-gold-light transition-all` with shimmer-gold class. Shows "Sending..." during submit (disabled).
- Success state: green CheckCircle icon, "Message Sent!" heading, "We'll be in touch within 24 hours" text, "Send Another" button to reset
- Error state: red AlertCircle icon, "Something went wrong" text, "Try Again" button

Use icons from lucide-react: CheckCircle, AlertCircle, ArrowRight.
