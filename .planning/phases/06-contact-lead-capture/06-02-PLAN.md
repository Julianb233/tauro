  <action>
Create `src/components/tour-booking-form.tsx` as a CLIENT component ("use client").

Import LeadPayload from `@/app/api/leads/route`, properties from `@/data/properties`.

**State:**
- `formData`: `{ firstName, lastName, email, phone, propertyId, preferredDate, preferredTime, agentPreference, message }` (all strings)
- `formState`: `"idle" | "submitting" | "success" | "error"`

**Props:** `{ preselectedPropertyId?: string }` -- if provided, set as initial propertyId value.

**Form fields:**

1. **Property selector** (select dropdown):
   - Label: "Which property are you interested in?"
   - Default option: "Select a property..." (value "")
   - Options from `properties` array: `${property.address} - ${formatPrice(property.price)}`
   - Value: property.id
   - Import `formatPrice` from `@/data/properties`

2. **Date picker** (input type="date"):
   - Label: "Preferred Date"
   - Min: today's date (use `new Date().toISOString().split('T')[0]`)

3. **Time slot selector** (select dropdown):
   - Label: "Preferred Time"
   - Options: "9:00 AM", "10:00 AM", "11:00 AM", "12:00 PM", "1:00 PM", "2:00 PM", "3:00 PM", "4:00 PM", "5:00 PM"

4. **Agent preference** (select dropdown):
   - Label: "Agent Preference (optional)"
   - Default: "No preference"
   - Options: agent names from properties or hardcoded list of Tauro agents

5. **Visitor info**: firstName, lastName (side by side), email, phone -- same styling as ContactForm

6. **Message** textarea: "Any special requests or questions?"

**Validation:** firstName, lastName, email, phone required. propertyId, preferredDate, preferredTime recommended but not blocking.

**Submit:** POST to `/api/leads` with:
```typescript
{
  type: "showing" as const,
  firstName, lastName, email, phone, message,
  propertyId,
  propertyAddress: selectedProperty?.address || "",
  preferredDate,
  preferredTime,
  agentPreference,
}
```

**Styling:** Same input styling as ContactForm: `bg-near-black border border-border rounded-lg px-4 py-3 text-foreground focus:border-gold focus:ring-1 focus:ring-gold`. Select elements need `appearance-none` with custom chevron or default browser styling. Submit button: gold with shimmer.

Success/error states same pattern as ContactForm.
