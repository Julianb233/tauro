Create `src/app/(site)/agents/[slug]/page.tsx` as a server component.

**generateStaticParams**: Map over agents array, return `{ slug: agent.slug }` for each agent. This enables static generation of all agent profile pages at build time.

**generateMetadata**: Accept `{ params: Promise<{ slug: string }> }`. Await params, look up agent by slug. Return:
- title: `${agent.fullName} - ${agent.title} | Tauro`
- description: First 120 chars of agent.bio + "Contact for expert real estate guidance."
- Return `{ title: "Agent Not Found" }` if agent not found.

**Default export**: Accept `{ params: Promise<{ slug: string }> }`. Await params, look up agent via getAgentBySlug. Call `notFound()` if not found. Render `<AgentProfile slug={slug} />` (or the client component wrapping the profile).

Note on Next.js 15 params: The params object is a Promise in Next.js 15 App Router — must await it before accessing slug.

Import from:
- `next/navigation` for notFound
- `@/data/agents` for agents, getAgentBySlug
- `@/components/agent-profile` for the profile component
